Commands frame/quantiles¶

New frame with Quantiles and their values.

GET /v1/commands/:id¶

Request¶

Route

POST /v1/commands/

Body

name:

frame/quantiles
arguments:

frame : Frame

<Missing Description>

What is being requested.

Content-type: application/json
