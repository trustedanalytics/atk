:doc:`VertexFrame <index>`  covariance
**************************************

------


.. function:: covariance(self, data_column_names)

    Calculate covariance for exactly two columns.


    :Parameters:

        **data_column_names** : list

        ..

            The names of two columns from which
            to compute the covariance.




    :Returns:

        : dict

        ..

            Covariance of the two columns.
    Notes
    -----
    This method applies only to columns containing numerical data.
